# Savan Vachhani Portfolio Website

A static portfolio website for Savan Vachhani, PhD - Data Consultant

## Project Structure

```
portfolio/
├── index.html       # Home page
├── work.html        # Work experience and expertise
├── cv.html          # Detailed CV
├── styles.css       # Custom CSS styles
├── assets/          # Images and other media
└── README.md        # This file
```

## Pages

### Home (index.html)
- Introduction and professional overview
- Personal branding and positioning

### Work (work.html)
- Professional background highlights
- Core expertise areas
- Industry experience

### CV (cv.html)
- Detailed curriculum vitae
- Full work experience at Baringa Partners and Anglo American
- Technical skills summary
- Education and qualifications
- Contact information

## Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **CSS3** - Custom styles and print styles
- **Responsive Design** - Mobile-friendly layout

## Styling

The website uses:
- Tailwind CSS utility classes for styling
- Yellow (#FACC15) as the primary brand color
- Gray tones for text and secondary elements
- Clean, professional design with consistent spacing

## Navigation

- Each page has a fixed navigation bar at the top
- Current page is highlighted with yellow background
- Links to other pages are always accessible

## Features

- ✓ Fully responsive design
- ✓ Print-friendly (navigation hidden when printing)
- ✓ Fast loading with Tailwind CDN
- ✓ Professional typography
- ✓ Easy to customize and extend
- ✓ No build process required

## Getting Started

1. Open any HTML file in a web browser
2. Use the navigation menu to switch between pages
3. The site is ready to deploy to any web server

## Customization

### Images
- Replace the placeholder emoji (👤) with an actual profile image by modifying the HTML
- Store images in the `assets/` folder

### Content
- Edit the text content directly in the HTML files
- Update personal information and details as needed

### Colors
- Primary color: Yellow (`bg-yellow-400`)
- Change colors by updating Tailwind classes
- Modify border colors and text colors as desired

### Fonts
- Currently using system fonts via Tailwind
- Can be customized by adding a custom font import

## Deployment

This static website can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any traditional web hosting

Simply upload the files to your hosting provider and your portfolio will be live!

## File Sizes

- HTML files: Lightweight (single-page files)
- No external dependencies required (Tailwind via CDN)
- Fast loading and excellent performance
